# Code of conduct

This project is a small open-source tool. We adopt the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) in spirit, and pare it to one operational rule:

**Be excellent to each other. Disagree with ideas, not people.**

In practice:
- Stay on-topic for the issue or PR you're responding to.
- Critique code or design, never identity.
- If someone shows up with a fix, lead with what works before what doesn't.
- If a thread is heating up, step away. The bug will still be there tomorrow.

## Enforcement

Behaviour outside these lines may result in:
1. A direct ask to revise the comment.
2. Removal of the comment by a maintainer.
3. A temporary or permanent ban from the project.

To report behaviour you find unacceptable: email **conduct@beemusic.co** with a link to the comment + context. We respond within 7 days.

Maintainers are not exempt — same rules, same accountability.
